belajar html dan css

html digunakan untu membuat tampilan website. css digunakan untuk mengatur tampilan website

materi html

  • h1 untuk judul
  • p untuk paragraf
  • img untuk gambar
  • table untul tabel
  • input untuk form
  • li untuk list atau daftar

materi css

  1. margin, padding, position untuk mengatur jarak antara elemen html
  2. display untuk mengatur elemen html agar tampil atau tidak tampil, mendatar(kolom), atau menurun(baris)
  3. style untuk mengatur font
